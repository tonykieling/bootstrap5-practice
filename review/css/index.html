CSS review

1- CSS Box Model

HTML elements can be considered boxes.

It contains:

  • Margin is a transparent area outside the border. It clears an area after the border.
  • Border goes around the padding and the content.
  • Padding is like the margin for the content.
  • Content yelds the content, text and images.

The full size of an element has to consider not only the content width and height, but also padding and border.

Margin counts to the space taken by an element, not specificaly for its total size.

Outline is a property that draws a border without modifying the size of the element - border does it.

Box Model Properties:

  • Height is relative to the vertical size of an element's content.
  • Width is the horizontal size of that element.
    • they change the size of block elements
  • Margin is the space around the border of an element. It is always transparent.
  • Border is the framework around the content. It can be colored and in diff forms such as solid and pointed.
  • Padding is the space around the content of an element.

    • Inline Elements

    They take up the same space as their content. It changes line when it is not possible to fit in that line.

    Examples are <a>, <span> and <strong>

    Block Elements

    They expand to the whole width of their container element, regardles of thei content width. It means that a block element always start a new line.

    Examples are <div>, <h1>, <p> and <section>

    Display Property

    It is used to change de default behavior of inline and block-level elements.

    Display: inline will turn a block element into a inline one.

    Display: inline-block will turn a block element into a inline one being possible to set the size.

    Display: block will turn a inline element into a block one.

    3- CSS Units

    1- Nesting selectors

    There are 4 types of combinators in CSS:

    1. Descendent selector --> space
    2. Child selector --> >
    3. Adjacent sibling selector --> +
    4. General sibling selector --> ~
    Example 1 - descendent selector: <space>
    Code HTML: 
              <div class="descendent-selector">
            <p>Test - with style</p>
          </div>
          <p>Test - without style</p><div class="descendent-selector">
            <p>Test - with style</p>
          </div>
    CSS: 
              .descendent-selector p {
            margin: 0;
            background-color: blue;
            color: darkred;
          }
     Output:

    Test - with style

    Test - without style

    Test - with style
    Example 2 - child selector: +
    Code HTML: 
              <div>
            <p>This is 'p' inside a 'div'<span> - and this is a 'span' inside that 'p'</span></p>
            <span class="span-temp">And this is a 'span' inside, child, a div.</span>
          </div>
    CSS: 
              div > .span-temp {
            color: gold;
            background-color: indigo;
          }
    Output:

    This is 'p' inside a 'div' - and this is a 'span' inside that 'p'

    And this is a 'span' inside, child, a div.
    Example 3 - adjacent sibling selector: ~
    Code HTML: 
              <div>
            <p>'p' line 1 inside 'div'</p>
            <p>'p' line 2 inside 'div'</p>
          </div>
          <p>'p' line 3 outside, right after, a 'div'</p>
          <p>'p' line 4 outside 'div'</p>
          <div>
            <p>'p' line 5 inside 'div'</p>
          </div>
          <p>'p' line 6 outside, right after, a 'div'</p>
    CSS: 
              div + p {
            background-color: sandybrown;
            text-align: center;
          }
    Output:

    'p' line 1 inside 'div'

    'p' line 2 inside 'div'

    'p' line 3 outside, right after, a 'div'

    'p' line 4 outside 'div'

    'p' line 5 inside 'div'

    'p' line 6 outside, right after, a 'div'
    Example 4 - general sibling selector
    Code HTML: 
              <div>
            It is a 'div'
          </div>
          <div class="m1">this is a 'div' which selector is a class called 'm1'</div>
          <p class="m1">It is a 'p' class = 'm1'</p>
          <h4>this is a plain 'h4'</h4>
          <h3 class="m2">a 'h3' class = 'm2'</h3>
    CSS: 
              div ~ .m1, .m2 {
            text-align: right;
            color: aquamarine;
            background-color: gray;
          }
    Output:
    It is a 'div'
    this is a 'div' which selector is a class called 'm1'

    It is a 'p' class = 'm1'

    this is a plain 'h4'

    a 'h3' class = 'm2'
    Example 5 - application of multiples classes
    Code HTML: 
              <div class="ex1 ex2">
            It is a example that applies 2 classes at the same time.
          </div>
    CSS: 
              .ex1 {
            color: white;
          }
          .ex2 {
            background-color: black;
          }
    Output:
    It is a example that applies 2 classes at the same time.
